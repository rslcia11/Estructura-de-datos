Enfoques y observaciones sobre la actividad 2.2. Gestión de la práctica y experimentación

El objetivo del código es implementar y demostrar el funcionamiento de dos algoritmos de ordenación (Bubble Sort y Selection Sort) y cómo buscar el mayor y menor elemento en un arreglo de datos.
Estos algoritmos son fundamentales para comprender los conceptos de ordenación de listas y búsqueda de valores en programación.

Algoritmos Implementados:
Bubble Sort: Este algoritmo recorre repetidamente el arreglo, comparando elementos adyacentes y los intercambia si están en el orden incorrecto. Se repite hasta que el arreglo esté completamente ordenado.
Selection Sort: Este algoritmo recorre el arreglo, selecciona el elemento más pequeño en el subarreglo no ordenado y lo intercambia con el primer elemento no ordenado. Luego repite este proceso para el siguiente subarreglo.
Búsqueda del mayor y menor elemento: Usamos las funciones integradas de Python max() y min() para encontrar el valor máximo y mínimo en un arreglo.
